# 👁️ Eye Disease Detection using Deep Learning

## 📌 Project Overview
This project uses Deep Learning (CNN + EfficientNet) to detect eye diseases like:
- Diabetic Retinopathy
- Glaucoma
- Cataract
- Normal

## 🚀 Features
- Image Upload via Web App
- Real-time Prediction
- Disease Info (Symptoms, Cause, Solution)
- High Accuracy Model

## 🧠 Technologies Used
- Python
- TensorFlow / Keras
- Flask
- OpenCV

## 📂 Project Structure
- app.py → Flask app
- create_model.py → Model training (EfficientNet)
- templates → HTML files
- static → CSS, JS

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py

Open browser:

http://127.0.0.1:5000/
📊 Model
Architecture: EfficientNetB0
Input Size: 224x224
📁 Dataset

Download from:

https://www.kaggle.com/
📷 Output
Displays disease prediction
Shows confidence score
Provides medical info
